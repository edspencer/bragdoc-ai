## Executive Summary

This research identifies **high-value keyword clusters** that BragDoc's target audience searches for. The goal is to create content that captures people looking for help with performance reviews, self-assessments, and career documentation.

### Key Finding
**People search for generic terms, not tech-specific ones.** "Software engineer performance review" only gets 30 searches/month, while "self evaluation examples" gets 4,400. Our content strategy should target generic terms while including tech examples.

## Content Strategy Recommendations

### 1. Hub Pages (Target Tier 1 Keywords)

Create comprehensive hub pages for the highest-volume generic terms:

- `/self-evaluation-examples/` - 4,400+ searches
- `/performance-review-examples/` - 5,400+ searches
- `/accomplishments-examples/` - 1,900+ searches
- `/peer-feedback-examples/` - 590+ searches
- `/360-review-guide/` - 2,400+ searches

### 2. Performance Review Encyclopedia (Company Pages)

For each of the 61 companies we've researched:

- `/performance-reviews/[company]/`
- Include system name in title for SEO (e.g., "Adobe Check-in Performance Review Guide")
- Target both "[company] performance review" AND system name keywords

**Priority companies by search volume:**
1. Adobe (880 for system name)
2. Amazon (260)
3. Microsoft (110 + 210 system)
4. Intel (10 + 140 system)
5. Meta (110)
6. Google (90)

### 3. Templates & Tools

Create interactive tools targeting high-intent template keywords:
- Self-evaluation template generator
- Performance review phrase builder
- Accomplishments tracker (the product!)

### 4. Seasonal Content

Publish/promote around review cycles:
- **December-January:** "Year end review" spikes to 5,400
- **June-July:** "Mid year review" spikes to 2,900
- **Quarterly:** "Quarterly review" steady at 590

### 5. Blog Content Topics

Based on keyword clusters, priority blog topics:
1. "How to Write a Self Evaluation" (880 searches)
2. "Self Appraisal Comments Examples" (3,600 searches)
3. "What to Say in Your Performance Review" (170 searches)
4. "How to Ask for a Promotion" (1,300 searches)
5. "Work Accomplishments Examples" (1,000 searches)
6. "OKR Examples" (1,900 searches)
7. "Professional Development Goals" (3,600 searches)
